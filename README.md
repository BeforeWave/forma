# Forma

Build plan-first AI skill suites with a verified shape.

## What this is

Forma is a plan-first AI skill creator. It ships three architectural layers:

- **Layer 1 — Instruction**: a pure SKILL.md plan-first creator that any compatible agent can read and follow to scaffold a plan-first skill suite for the user's context.
- **Layer 2 — Verification**: a programmatic verifier that checks whether an agent-generated skill suite conforms to the plan-first methodology.
- **Layer 3 — Generation**: a programmatic creator with a business-injection layer that produces deterministic, multi-variant, multi-target skill suites.

The methodology that underpins all three layers is **plan-first**: an executable plan converges Goal, Scope, Approach, Validation, and Plan Strategy before any implementation begins; each task carries its own acceptance and validation contract; evidence lives next to the plan it proves.

## Status

Bootstrap. The source tree is intentionally empty at this stage. Subsequent issues will populate the methodology source, the verifier, and the generator. See `plans/issue-<id>/` for the current planning and execution state.
