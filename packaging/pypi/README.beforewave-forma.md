# beforewave-forma

`beforewave-forma` is a compatibility install-name package for Forma's Python
CLI.

New installs should prefer `forma-cli`:

```bash
pipx install forma-cli
forma --help
```

This package ships the same `forma` import package, console command, workflow
compiler, runtime assets, and verifier.

## Install

```bash
pipx install beforewave-forma
forma --help
```

Temporary run:

```bash
uvx beforewave-forma forma --help
```

The source repository is <https://github.com/BeforeWave/forma>.
